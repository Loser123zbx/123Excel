123Excel (原名123XLS)
介绍
这是一个帮助教师群体运用Excel表格的工具，无需冗长的函数，有着方便的GUI界面

软件架构
Windows x64适用 Windows x32可能可用 不适用Mac，Linux等 这是因为国内教师群体用Windows居多

安装教程
下载安装程序，不要下载zip包，除非你有python运行环境
运行安装程序
等待安装完毕
使用说明
打开软件
看软件内的控件名称即可知道其作用，在此不多赘述
